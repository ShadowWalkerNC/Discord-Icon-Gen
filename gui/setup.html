Bot Credentials

Enter your bot token and application ID from the Discord Developer Portal. Your token is validated live before you can continue.

Bot → Token in the Developer Portal. Never share this with anyone.
General Information → Application ID in the Developer Portal.
Not validated

Platforms & API Keys

Toggle which external platforms Sigil should connect to. Only active platforms need a key — inactive ones are completely skipped. Keys you provide here are saved to your environment.

ℹ️These are your own API keys. Sigil never shares or proxies them through any third-party service.

Feature Packages

Choose which feature groups to enable. You can toggle any package later from the Packages page.

Channels & Roles

Map each feature to a Discord channel or role. Enable Developer Mode in Discord (Settings → Advanced), then right-click any channel or role to Copy ID. All fields are optional.
